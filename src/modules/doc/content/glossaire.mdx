# Glossaire

Les termes métier employés dans toute cette documentation et dans l'application. À lire en premier si vous découvrez France Chaleur Urbaine.

## Acteurs et rôles

| Terme | Définition |
| --- | --- |
| Demandeur | Personne qui dépose une demande de raccordement : un **particulier** ou un **professionnel**. Voir [Rôle : particulier et professionnel](/admin/doc/role-demandeur). |
| Gestionnaire | Exploitant d'un réseau de chaleur (ENGIE, Dalkia, Coriance…) qui traite les demandes de ses réseaux. Voir [Rôle : gestionnaire](/admin/doc/role-gestionnaire). |
| Collectivité | Commune ou EPCI qui suit les demandes de son territoire. Voir [Rôle : collectivité, ALEC, CCRT](/admin/doc/role-collectivite). |
| ALEC | Agence locale de l'énergie et du climat : même rôle qu'une collectivité, à l'échelle d'un territoire. |
| CCRT | Chargé de mission compétent sur un territoire : même rôle qu'une collectivité. |
| Admin | Équipe France Chaleur Urbaine, qui pilote l'ensemble. Voir [Rôle : admin](/admin/doc/role-admin). |

## Réseaux

| Terme | Définition |
| --- | --- |
| Réseau de chaleur | Ensemble de canalisations souterraines qui distribuent de la chaleur (souvent renouvelable) depuis une centrale vers des bâtiments. |
| Réseau de froid | Équivalent pour la distribution de froid. |
| Réseau en construction | Réseau prévu ou en cours de travaux, pas encore en service. |
| Réseau classé | Réseau bénéficiant d'un classement qui rend le raccordement **obligatoire** dans son périmètre. |
| PDP | Périmètre de développement prioritaire : la zone d'un réseau classé où l'obligation de raccordement s'applique (amende possible jusqu'à 300 000 €). Voir [Test d'éligibilité](/admin/doc/eligibilite). |
| Tracé | Le parcours géographique des canalisations d'un réseau ; sert à calculer la distance d'une adresse au réseau. |
| EnR&R | Taux d'énergies renouvelables et de récupération dans la chaleur produite par le réseau. |
| Contenu CO2 ACV | Émissions de CO2 par kWh livré, en analyse de cycle de vie. |
| Identifiant SNCU | Identifiant national d'un réseau (utilisé dans les exports et les fiches réseau). |

## Demandes et éligibilité

| Terme | Définition |
| --- | --- |
| Demande | Demande de mise en relation en vue d'un raccordement, déposée par un demandeur. Voir [Cycle de vie d'une demande](/admin/doc/cycle-de-vie-demande). |
| Raccordement | Le fait de relier un bâtiment à un réseau de chaleur. |
| Éligibilité | Le fait qu'une adresse soit suffisamment proche d'un réseau pour être raccordable. Voir [Test d'éligibilité](/admin/doc/eligibilite). |
| Chauffage collectif / individuel | Collectif : une chaufferie unique pour tout le bâtiment (raccordement plus simple). Individuel : un chauffage par logement. La distinction conditionne plusieurs règles (statut, relances). |
| Affectation | Le rattachement d'une demande à un réseau, automatique à la création puis modifiable. |
| Réaffectation | Le changement de réseau d'une demande, proposé par un gestionnaire ou une collectivité et arbitré par l'équipe FCU. Voir [Réaffectation des demandes](/admin/doc/reaffectation-demandes). |
| Validation | Étape par laquelle l'équipe FCU rend une demande visible des gestionnaires. |
| Relance | Message automatique de suivi, adressé aux gestionnaires ou aux demandeurs. Voir [Relances et notifications](/admin/doc/relances-notifications). |

## Sources de données

| Terme | Définition |
| --- | --- |
| BAN | Base Adresse Nationale : convertit une adresse en coordonnées géographiques (géocodage). |
| BDNB | Base de Données Nationale des Bâtiments : fournit le nombre de logements et d'autres caractéristiques d'un bâtiment. |
| ADEME Connect | Le CRM de l'ADEME, où sont synchronisés les contacts des utilisateurs. |
| Airtable | Ancien outil de gestion, encore utilisé pour certaines synchronisations de données réseau. |
| Publicodes | Moteur de règles servant aux calculs réglementaires et financiers du comparateur de coûts. |
